# OS--Real-Time-Process-Monitoring-Dashboard

## Features
- Real-time CPU & Memory monitoring
- Live charts
- Running process list
- WebSocket-based updates

## Tech Stack
- Next.js
- Node.js + Express
- Socket.IO
- Recharts

## Run
1. Start backend:
   cd backend && node server.js

2. Start frontend:
   cd frontend && npm run dev

## Output
Dashboard updates every 2 seconds.
